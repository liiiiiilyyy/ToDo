# 📋 ToDo Веб

Пользователь открывает веб-страницу в браузере и видит красивый, чистый интерфейс. Чтобы добавить новое дело, достаточно написать текст в поле ввода и нажать кнопку "Добавить". Задача сразу появляется в списке.

Каждое дело можно:

Отметить как выполненное — нажать на само дело или на кнопку ✅, после чего оно зачёркивается и становится прозрачным

Удалить — если дело больше не актуально, его можно убрать из списка нажатием на кнопку 🗑

Внизу страницы автоматически обновляется статистика, которая показывает:

Общее количество дел

Сколько дел уже выполнено

Это помогает пользователю видеть свой прогресс и мотивирует двигаться дальше.

## 🎯 Проблемы, которые решает проект

### Для кого (целевая аудитория):
- Студенты, которые забывают про дедлайны
- Работающие люди, у которых много задач
- Все, кому нужно простое приложение для планирования

### Какие проблемы решаем:

| Проблема | Как решает проект |
|----------|-------------------|
| Люди забывают важные дела | Список дел всегда перед глазами в браузере |
| Существующие приложения сложные | Простой интерфейс — открыл и пользуешься |
| Нет бесплатных удобных трекеров | Полностью бесплатно, работает локально |
| Трудно отслеживать прогресс | Визуальная отметка выполненных дел + статистика |

### Пример использования:
1. Открываешь сайт
2. Добавляешь дело: "Купить продукты"
3. Делаешь дело → нажимаешь ✅
4. Удаляешь ненужное → 🗑
