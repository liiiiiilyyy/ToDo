# 📋 ToDo Веб

Простой и красивый веб-трекер дел. Добавляй, отмечай выполненные и удаляй задачи прямо в браузере.

---

## 🎯 Проблема и целевая аудитория

### Проблема
Люди забывают о важных делах. Существующие приложения сложные, требуют регистрации и установки.

### Целевая аудитория
- Студенты, которые забывают про дедлайны
- Работающие люди, у которых много задач
- Все, кому нужно простое приложение для планирования

---

## 🚀 Решение

Простое веб-приложение для списка дел, которое работает **без установки и регистрации**.

### Возможности
- ✅ Добавление дел с датой, временем и приоритетом
- ✅ Отметка выполненных задач
- ✅ Удаление задач
- ✅ Фильтрация (все / активные / выполненные)
- ✅ Статистика (всего / активных / выполнено)
- ✅ Автоматическое определение просроченных задач
- ✅ Сохранение данных в браузере (localStorage)
