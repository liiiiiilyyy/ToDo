# 📋 ToDo Веб — Умный список дел

Простой, красивый и удобный веб-трекер задач. Добавляй дела, ставь дедлайны, отмечай выполненные и следи за прогрессом. **Всё в одном файле!** Никаких установок, регистраций и серверов.

---

## 🎯 Проблема и целевая аудитория

### Проблема
В современном мире люди забывают до 40% запланированных дел. Существующие приложения сложные, требуют регистрации, установки и денег. Нужен простой инструмент, который всегда под рукой.

### Целевая аудитория
- **Студенты** — забывают про дедлайны, нет денег на платные приложения
- **Фрилансеры** — много проектов, важно не пропускать сроки
- **Занятые родители** — нужно планировать дела всей семьи
- **Все, кто хочет организовать свою жизнь** — планирование покупок, запись идей, отслеживание привычек

---

## 🚀 Наше решение

**ToDo Веб** — это веб-приложение, которое работает **без установки, регистрации и сервера**. Открыл браузер и пользуешься!

### Как это работает

Пользователь открывает веб-страницу в браузере и видит красивый, чистый интерфейс. Чтобы добавить новое дело, достаточно написать текст в поле ввода и нажать кнопку "Добавить". Задача сразу появляется в списке.

Каждое дело можно:
- **Отметить как выполненное** — нажать на само дело или на кнопку ✅, после чего оно зачёркивается и становится прозрачным
- **Удалить** — если дело больше не актуально, его можно убрать из списка нажатием на кнопку 🗑

Внизу страницы автоматически обновляется статистика, которая показывает:
- Общее количество дел
- Сколько дел уже выполнено

Это помогает пользователю видеть свой прогресс и мотивирует двигаться дальше.

## 🚀 Как запустить

### Способ 1: Локально (РЕКОМЕНДУЕТСЯ)
1. Скачай файл `index.html`
2. **Дважды кликни** по нему
3. Откроется в браузере — пользуйся!

### Способ 2: Через GitHub (если открывается)
Перейди по ссылке: https://raw.githubusercontent.com/liiiiiilyyy/ToDo/refs/heads/main/index.html


### Способ 3: Скачать архив
1. На GitHub нажми **Code** → **Download ZIP**
2. Распакуй архив
3. Открой `index.html` в браузере
