## 🚀 Наше решение

**ToDo Веб** — это клиент-серверное веб-приложение для управления списком дел. 

### Как это работает

**Клиентская часть (фронтенд):**
Пользователь открывает веб-страницу в браузере и видит красивый, чистый интерфейс. Чтобы добавить новое дело, достаточно заполнить форму и нажать кнопку "Добавить". Задача сразу появляется в списке.

Каждое дело можно:
- **Отметить как выполненное** — нажать на кнопку ✅, после чего оно зачёркивается
- **Удалить** — если дело больше не актуально, нажать на кнопку 🗑

Внизу страницы автоматически обновляется статистика, которая показывает:
- Общее количество дел
- Количество активных дел
- Сколько дел уже выполнено

**Серверная часть (бэкенд):**
Все данные хранятся в базе данных MySQL. Когда пользователь добавляет, изменяет или удаляет дело, клиент отправляет запрос на сервер (API), который обрабатывает его и сохраняет изменения в базе данных. При обновлении страницы все дела загружаются из базы и отображаются пользователю.

Это помогает пользователю видеть свой прогресс и мотивирует двигаться дальше.

## 🛠 Архитектура и стек технологий

### Клиентская часть (Фронтенд)
- **HTML5** — структура страницы
- **CSS3** — красивый дизайн с градиентами и анимациями
- **JavaScript (ES6)** — логика приложения, отправка запросов на сервер

### Серверная часть (Бэкенд)
- **PHP** — обработка запросов, работа с базой данных
- **MySQL** — реляционная база данных для хранения задач
- **REST API** — обмен данными между клиентом и сервером

### Инфраструктура
- **OpenServer / XAMPP** — локальный сервер для разработки
- **Git** — управление версиями
- **GitHub** — хранение репозитория и командная работа

## 🚀 Как запустить

### Способ 1: Через OpenServer (РЕКОМЕНДУЕТСЯ)

1. Установи [OpenServer](https://openserver.ru/download/)
2. Запусти OpenServer
3. Создай папку `todo.local` в `C:\OpenServer\domains\`
4. Скопируй файлы проекта (`index.html`, `api.php`) в эту папку
5. Открой в браузере: `http://todo.local`
